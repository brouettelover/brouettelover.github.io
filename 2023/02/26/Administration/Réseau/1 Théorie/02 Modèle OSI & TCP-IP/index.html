Modele OSI et TCP/IP

Description

Le modele OSI est un modele de representation figuratif representant les differentes couches du reseau.
Ce modele est purement theorique et permet une meilleure comprehension general des protocoles de communication.

Elle est souvent compare au modele TCP/IP qui represente le schema reel de la connectivite reel des reseaux actuels.

INSERER COMPARATIF IP OSI

1 Couche physique

Ensemble de moyens permettant de gerer les connexions physique (materiels) pour la transmission de bits
Unite : bit

2 Couche liaison de donnees

Decrit les echanges de trames de donnees entre peripheriaque sur un support commun (reseau local).
Unite : Trame

3 Couche Reseau

Echange des parties de donnees entre peripherique finaux sur le reseau ( Internet ).
Unite : Paquet

4 Couche Transport

Segmente, reassemble, transfere les donnees de communications sur le reseau entre les peripheriques finaux.
Unite : Segment

5 Couche Session

Fournit des services a la couche presentation pour organiser et gerer l echange de donnees.
Unite : /

6 Couche Presentation

Une representation commune des donnees transfere entre des services de couches applications
Unite : /

7 Couche Application

Connectivite entre les individus via des reseaux de donnees
Unite : Donnee
