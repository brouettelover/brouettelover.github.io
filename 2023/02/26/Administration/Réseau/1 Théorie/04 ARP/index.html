ARP

Brouettelover 2023-02-26 17:32:31
Categories： Réseau > Administration > Couche 2 Tags： Fonctionnement Protocole ARP

Description

Une table ARP est une table d ip regroupant des adresses logiques (IP) a des adresses physique (MAC).

Un équipement se réfère à sa table ARP interne pour l’envoie des données vers les hotes correspondants aux IP. S’il ne connait
pas l’adresse MAC associé à l’IP, il propage une trame afin d’intéroger les autres équipements afin de trouver l’hôte possédant l’adresse IP correspondante.

Construction d’une table arp

1 ere manière

L’équipement surveille le trafic sur le réseau local. Quand une trame est émise sur le reseau et qu’elle passe par un noeud.
L’équipement enregistre l’ip source et l’adresse mac dans sa table arp.

Horodatage :

La table réinitialise les données qu’elle contient sur une période donnée

2nd manière:

Il est possible d’ajouter manuellement une entrée dans la table ARP

Comment sortir du reseau Local ?

La source utilise la passerelle par défaut et le protocole ARP afin de transférer un paquet sur un autre reseau.
