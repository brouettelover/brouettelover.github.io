Conception de réseaux - Les types de commutateurs

Brouettelover 2023-02-26 17:32:31

Les différentes types de commutateurs

Commutateur à configuration fixe

Commutateur les plus bons marchés, leur taille est représenté par une unité de stack appelé “u”. Ces commutateurs sont utilisés pour les petits réseau, ils ne permettet pas une grande évolutivité.

Commutateur modulaire

Ils sont les commutateurs les plus coûteux du marché et se présentent sous la forme d’un chassi comportant plusieurs commutateurs afin d’assurer une grande évolutivité.

Commutateur Empilable

Les commutateurs empilables sont une solution alternative aux commutataires modulaires, leur coût est moindre. Il est possible d’en empiler jusqu’à 9 via la technologie cisco StackWire. Ces commutateurs ne sont représenter que par une IP sur le réseau et constitue une sorte de “grand” commutateur.

caractéristiques :

  • 1 seule adresse IP de gestion (qui n'est pas liée à un commutateur).
  • Table de commutation partagée.
  • Représente un seul switch pour le Spanning Tree Protocol.
  • L'ajout de commutateurs n'augmente pas le diamètre réseau.

Les fonctions les plus utiles en fonction des couches du réseau hiérarchique

La couche accès

rappel de la couche d’accès : couche permettant aux utilisateurs finaux d’intéragir avec le réseaux.

Pour la couche accès, il est recommandé aux commutateurs d’avoir les fonctionnalités suivantes :

  • VLANs
  • Fast Ethernet/Gigabit Ethernet
  • Port security
  • Link aggregation
  • Power over Ethernet (PoE)
  • Quality of Service (QoS)

La couche distribution

rappel de la couche de distribution : couche servant à mettre en communications les différents utilisateurs du réseaux.

Pour la couche distribution, il est recommandé aux commutateurs d’avoir les fonctionnalités suivantes :

  • Switch Layer 3
  • Gigabit Ethernet/10 Gigabit Ethernet
  • High forwarding rate
  • Link aggregation
  • Security policies/Access Control Lists
  • Redundant components
  • Quality of Service (QoS)

La couche coeur

rappel de la couche coeur : fourni l’accès à l’extérieur du réseau de l’entreprise

Pour la couche coeur, il est recommandé aux commutateurs d’avoir les fonctionnalités suivantes :

  • Switch Layer 3
  • Gigabit Ethernet/10 Gigabit Ethernet
  • Very high forwarding rate
  • Link aggregation
  • Redundant components
  • Quality of Service (QoS)
